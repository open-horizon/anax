# Configuring policies
{: #configuring_policies}

Policy-based deployment enables containerized workloads (services) to be deployed to edge nodes running the {{site.data.keyword.horizon}} agent. Policies define the properties and constraints that govern workload deployment and node management.
{:shortdesc}
